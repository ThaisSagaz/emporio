Esta é a tela onde ficam os Usuários do sistema. Funciona basicamente como as outras tabelas,também possui as opções de Atualizar,Visualizar e Apagar,




Para obter mais informações sobre o usuário clique em visualizar.





Para adicionar um novo usuário, clique em "Novo", preencha o formulário e selecione a opção "Criar" como mostra a imagem abaixo.









Para atualizar os dados do usuário selecione o botão "Atualizar" e após preencher devidamente, clique em "Criar".





Para excluír um usuário é da mesma forma que as outras telas anteriores







Agora vamos para a tela Tipo de Produto,onde será especificado quais são os produtos vendidos na loja.









Para renomear o tipo de produto,clique em "Atualizar"







Para criar um novo tipo de produto, clique em "Novo"







Para excluir um tipo de produto,selecione a opção excluir,você irá para a tela logo a baixo:






Vamos para a tela de Cliente







Para adicionar um novo cliente, selecione a opção "Novo" e preencha o formulário corretamente como mostra a imagem abaixo:









Selecionando a opção "Visualizar" verá mais informações sobre o cliente como a imagem abaixo:






A opção Apagar é sempre a mesma,como na imagem abaixo:






Para alterar os dados do cliente,basta clicar no botão "Atualizar" que será redirecionado para a página onde mudará os dados no formulário como segue a imagem abaixo :







Agora iremos para a tela de Produtos:







Clicando em visualizar você irá ver mais informações como a imagem abaixo:








Para excluir um produto,selecione a opção "Apagar" como na imagem abaixo :








Clicando em novo você cadastrará o produto preenchendo os campos corretamente, em tipo de produto você selecionará se é uma torta doce,salgada,cupcake ou doce,em baixo temos uma pergunta "Fatia?" se for uma fatia selecione a opção "SIM" e em baixo terá a opção de escolher a torta que ja foi cadastrada no sistema, Se não for uma fatia, o campo "Selecionar torta" estará desabilitado, basta selecionar o preço da torta no próximo campo e o tamanho dela como mostra a imagem a seguir.
