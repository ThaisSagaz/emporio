No menu de vendas existem duas opções, a primeira é a entrada de produtos, nessa tela será exibido os item que estão no estoque de vendas.




Clicando no botão "Novo"  poderá adiconar outros produtos no estoque de venda, lembrando que os produtos inseridos aqui, só exibirão na venda se tiverem com a situação de "Disponível"




Para fatiar uma torta é necessário clicar no botão "Fatiar torta" escolher qual a torta que será fatiada, caso as fatia possua preço promocional, favor digitar o valor correspondente.






Para atualizar o produto do estoque venda, clique no botão atualizar e faça as alterações desejaveis.






Abaixo está uma tela para realizar venda.

Escolha o produto logo depois aparecerá um campo com a quantidade escolhida(lembrando que essa quantidade é o limite disponível no estoque) e o valor do produto escolhido. Clicando em inserir. será adicionando numa tabela ao lado que será o carrinho de compras, caso o cliente queira remover algum item do carrinho, só clicar a imagem ao lado de cada intem. Finalizando a inserção de intens no carrinho, favor conferir o valor total e clicar no botão "Efetuar venda".
